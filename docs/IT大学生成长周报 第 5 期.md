### 怎么做职业规划？

[点此查看原文](https://mp.weixin.qq.com/s/C1PjNPbGQ5Ab59t29Fz_nA)

做职业规划，取决于自己的自信程度和格局大小。

>如果今天有一个人，已经60岁退休了，他回顾自己的历史，知道自己在50岁的时候创立了一家年收入30亿的公司。

>他可能会想，我在50岁的时候要成就一个30亿的公司，但我其实可以成就100亿的。我当时最缺的是什么呢？

>我最缺的是理工科的基础，很多事情我不扎实。
>
>所以这个时候他可能会干什么？
>
>他可能在大学四年就会好好地学理工科。
>
>虽然学理工科可能不挣钱，但他心中不怕，因为他知道自己在50岁的时候就能赚30亿了。
>
>于是，到大学毕业找工作的时候，他就去找了一个非常底层的，叫做精算师的工作。他接下来走的每一步，都是为了积累50岁时所需要的能力，虽然可能并不赚钱。
>
>所以你看，他为什么能有这么大的定力？
>
>是因为他自信，他在50岁的时候能赚30亿，这种自信来自于什么？
>
>来自于他看到了历史。
>
>那么，当你站在18岁的节点，如果让自己来导演这一生，你觉得自己能在50岁，40岁，30岁的时候做些什么？
>
>你的格局越大，你就会做更长期主义的规划。

另外，职业规划要分三个阶段去做：

>**在25-35这个阶段，是拼体力和智力的阶段。**
>
>我自己刚参加工作的时候，就是拼命加班。为什么？
>
>因为我要靠体力去换取经验的增长、能力的增长，和资源的增长。
>
>**到了35-45的阶段，是拼智力和资源的阶段。**
>
>这个时候，你去和25岁的年轻人拼体力，你已经拼不过了。
>
>但是，你拥有年轻人没有的经验和资源，以及对事情的判断力。
>
>所以这个阶段，你靠智力和资源挣钱。
>
>**到了45-55的阶段，是拼人脉和资本的阶段。**
>
>这个时候，你经验积累的速度开始放慢了，你的知识结构也开始稳定了，你的智商其实也慢慢衰弱了。
>
>但你已经积累了很多的人脉，也积累了不错的资本。
>
>所以这个阶段，你靠人脉和资本挣钱。

### 为什么 Go 有两种声明变量的方式，有什么区别，哪种好？

在 Go 中，一共有 2 种变量声明的方式，各有不同的使用场景。

分别是：

- 标准变量声明（Variable declarations）。
- 简短变量声明（Short variable declarations）

***

用var的标准声明会给每个变量一个类型和初始值，短变量不会。

**分组声明：**

多个变量可以通过var来进行分组声明，短变量不支持。

```go
var (
 i    int
 u, v, s = 1.0, 2.0, "脑子进煎鱼了"
)
```

**变量的初始值指定：**

标准变量定义可以只声明，不主动定义初始值（默认会给零值）。

短声明则不行。

**局部变量，区分作用域：**

经常有一些局部变量声明，用短变量声明会更好。

**重新声明变量：**

拿下面的代码来说，短变量声明是可以不断重新声明变量的，会进行覆盖。而标准声明不行。

```go
var name = "煎鱼.txt"

fi, err := os.Stat(name)
if err != nil {
    log.Fatal(err)
}

data, err := ioutil.ReadFile(name)
if err != nil {
    log.Fatal(err)
}
...
```

### 人生商业模式：如何过好这一生？

[点此查看原文](https://mp.weixin.qq.com/s/FghzpsUhSxpfpm-OKhPToQ)

人生商业模式=能力 * 效率 * 杠杆

![image-20220113144213963](https://img-blog.csdnimg.cn/img_convert/2912b91a2e0ed3e49714f5ba34734cc1.png)

***

**能力：**

对一个人最有价值的能力不是**演讲能力**、**学习能力**、**沟通能力**、**赚钱能力**，而是**获得能力的能力**。

也就是你怎么只用2年的时间，获得别人5年的能力？

那怎么才能具备这样的能力呢？答案就是**加班**。换一个词叫**勤奋**。

别人一天工作8小时，我一天工作16个小时，我是不是就有可能用2年获得别人5年的能力呢？ 

那么什么是勤奋？现在很多公司是996，早9晚9，工作6天。甚至有一些公司是711，早7晚11点，全年无休。

对自己要求711的人，996就是放假，这就是勤奋。

但是，勤奋还远远不够。——需要**可怕的勤奋**。

什么是可怕的勤奋？ 拿微软来讲，微软早上，很多人来公司第一件事是把员工卡仍在一张床上占位置，不是为了午休，而是为了彻夜睡在办公室，在微软干到凌晨都不好意思走，因为即便到了那个点，整个办公室还都是人。

抢不到床有睡袋可以睡到会议室的地上，然后起床洗漱，继续工作。

这就是**可怕的勤奋**。

但是可怕的勤奋就够了吗？也还远远不够，需要**高效而可怕的勤奋**。

就是**刻意练习**，让自己处在高效的反馈机制和困难之中。

**当然，这一切是有前提的：**

第一，你真的想要拥有获得能力的能力。

第二，确保所有的勤奋，都在你身体和家庭的承受范围之内。

***

**效率：**

拥有了能力，还要提高做事的效率——把一个小时用出三个小时的效果。

有三个方法论来支撑——选择、方法、工具。

**选择：**

**真正能够提高你效率的方法，不是如何从17分钟里省出17秒，而是如何用17分钟省出17个小时。**

什么意思？

就是你要在这17分钟里做出一个决定，接下来要花费17小时做的事情，到底值不值得做？

这就叫做选择。

哪些事情是你实现人生目标所必须要做的？

哪些事情是对你的人生目标帮助不大的？

哪些事情是你哪怕失去现有条件也一定要完成的? 

选择，是提高效率的第一要义。

**方法：**

比如，有一家的孩子丢了，妈妈是转发朋友圈？是张贴寻人启事？

后来妈妈把一张照片发在了微博上，这是很正常的，不正常的是——发了一张这个小男孩穿比基尼的照片。

然后引起了广泛的传播，这就是**高效的方法**。

**工具：**

比如，白板。

我最经常的工作状态，就是站在巨大的白板前，把想法像小石子一样，扔到知识储备的湖面上，迅速把激起的浪花、涟漪记录在白板上。

然后，我会退后半米，静静地看着这些想法舒展、连接、形成结构，感受创造带来的喜悦和成就感。

但除了喜悦和成就感，白板还解决了我思考过程中，三个非常实际的问题：

> 第一，相对于Word，Excel，PPT，它能把我从“结构化的思维”里解放出来，随心所欲地思考。
>
> 第二，相对于A4纸，它能把我从“有边界的思维”里解放出来，在广阔的空间里舒展、连接。
>
> 第三，相对于翻页纸（Flipboard），它能把我从“不能错的思维”里解放出来，想到就写，写错就擦，擦了再来。

***

**杠杆：**

一个人的效率再怎么提升，每天也只有24小时，永远是有上限的。

所以需要借助杠杆——**团队杠杆**、**产品杠杆**、**资本杠杆**、**影响力杠杆**

团队杠杆：团队成员，用团队来复制做大，是最基础的杠杆，要学会娴熟使用。

产品杠杆：印刷业替代了人工誊写。

影响力杠杆：

能接触到一些最有价值的产品吗？

你能找到一个最好的团队吗？

你能让别人真的相信你，并且给你投资吗？

这就叫做影响力。

**可是怎么样才能获得更大的影响力呢？**

你需要三种能力：演讲能力、写作能力、以及建立人脉的能力。

演讲和写作是两个大规模杀伤性武器，如果你想扩大自己的影响力，你就要持续训练。

那建立人脉的能力呢？

关于人脉，你需要记住一句话：**人脉，不是那些能够帮到你的人，而是那些你能帮到的人。**

**杠杆能帮你获得巨大的成功，但是使用杠杆也是有前提的：**

你必须先有强大的能力内核。

记住，所有的杠杆，不论是团队、产品、资本、还是影响力，它们的作用都是复制放大。

复制放大，不必然导致成功。

如果你的能力内核很强大，通过使用杠杆，你会更快地获得成功。

但是，如果你的能力内核很虚弱，杠杆，只会加速你的失败。

## 本周主题——大班制度和师徒制度

在我们学校中会有一些课外兴趣小组，而我所处的组织，我们都称它为三月，因为它是三月诞生的。

今天想和你聊聊**大班制度**和**师徒制度**中我的感受。

在我人生的经历中经历过两种教育模式，一种是**大班制度**，一种是**师徒制度**。

所谓**大班制度**，人数一般会比较多，比如20/30/40这个体量，平时我们在小学，初中，高中中一般是这样的教育模式，上层者负责制定一个大的战略和方针，下面的人跟着这个方向往前走就可以了，有问题反馈给上级，上级进行解决或调整。

我觉得**大班制度**看的是管理的策略，对上层领导的要求不会特别高。

而**师徒制度**，可能只有2个人，5个人，10个人的体量，人数比较少，大家对于这种制度的体会可能更多的是在我们的传统文化中听说过。 它一般在研究生以上的学历中，才是这种教育模式，而我有一部分时间在中专度过，中专技校中，就有一部分师徒制的元素——手把手教你技能如何运用，但不是所有人都有这个机会，能够每天跟着老师耳濡目染的学习东西。

这个制度主要看的是**人**，人高明了，组织内的氛围就好，师父在哪儿，哪儿才能起来，师父走了，组织也会随之消解。

我的人生中经历过这两种制度，我个人比较喜欢**师徒制度**的方式去带人，它更具体，更细腻，也更直达人心，可达对方的心，也可达自我的心，只是会累一些。

比如说组织开始时都有**自我介绍**的环节，大班制度往往是蜻蜓点水，一掠而过，更多的还是希望大家在未来的陪伴中慢慢互相了解。  而我在这个环节中，常常不希望会那么无聊的度过， 我会在这个过程中，告诉大家如何**自我介绍**，现在有什么问题，对别人是什么感受，以及如何关注别人的话，看到别人话背后的含义。

**师徒**的模式最开始，就是交心的，坦诚的，它主要是靠上层者的个人能力和个人魅力支撑着，不仅仅是传授知识，而是传授习惯，思维，生活中的细节，感受，帮助下面的人建立一个健全的人格。

常常，如果想做一个好的老师，需要不断的精进自己，对自己的要求要高，要不然可能无法很好的进行指导，每个人都有不同的特点，每个人所需要的也不一定相同， 所以对老师的要求会很高，你可能需要了解10分的道理，每个人才能从中各取一分的养分。
